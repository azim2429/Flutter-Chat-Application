## Setup

  ##### Clone the repository
```bash
git clone https://github.com/azim2429/Flutter-Chat-Application.git
```
  ##### Move to the desired folder
```bash
cd \chat-app
```

#### To install dependencies
```bash
flutter pub get
````

  ##### To run the app, simply write
```bash
flutter run
```
